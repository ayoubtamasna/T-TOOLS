## 🎯 What is T-TOOLS?

T-TOOLS is a **free and open source** professional infrastructure diagnostic tool built with Python. Inspired by CPU-Z but designed for infrastructure professionals, it provides 9 powerful modules for complete system analysis.

---

## 📦 Modules

| # | Module | Description |
|---|--------|-------------|
| 🖥️ | **General Hardware** | CPU, RAM, GPU, Motherboard, Disk info with brand logos |
| 📊 | **Live Graphs** | Real-time CPU & RAM graphs updated every second |
| 🏥 | **Health Score** | System health score /100 with detailed breakdown |
| 🔔 | **Smart Alerts** | Custom threshold alerts for CPU & RAM |
| 🔌 | **Port Scanner** | Scan open/dangerous ports on any IP |
| 🔐 | **WPS Scanner** | WiFi network scanner + WPS vulnerability detection |
| 🔒 | **Cyber Security** | Firewall, antivirus, connections, shared folders |
| 💻 | **IT Information** | OS details, services, startup programs, events |
| ☁️ | **Cloud / Infra** | Network interfaces, DNS, routing, virtualization |

## 🚀 Quick Start

### Option 1: Download EXE (Recommended)
```bash
# No installation needed!
# Just download and run:
```
**[⬇️ Download T-TOOLS v1.1.0](https://github.com/ayoubtamasna/t-tools-releases/releases/download/v1.1.0/T-TOOLS.exe)**

> ⚠️ If Windows SmartScreen appears: Click **"More info"** → **"Run anyway"**
> This is normal for new software. [View VirusTotal Report](https://www.virustotal.com/gui/file/e380dbd8bf2c9a52b35e41000cc7bd705caed452ccae67d91b474d800b58ccbc)

### Option 2: Run from Source
```bash
# Clone the repository
git clone https://github.com/ayoubtamasna/T-TOOLS.git
cd T-TOOLS

# Install dependencies
pip install psutil wmi customtkinter requests scapy reportlab pillow pythoncom matplotlib wifi

# Run
python main.py
```

### Option 3: Build EXE yourself
```bash
pip install pyinstaller
python -m PyInstaller --onefile --windowed --name "T-TOOLS" --icon=assets/logos/favicon.ico main.py
```
